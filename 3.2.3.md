- Improved the positioning of untrim visuals

- Previous update: Run `/trigger Trims_Menu.BLUE set 10` to view your current combat timer

- Snout's gold ingots won't activate Gamble if there is a Piglin nearby

- Flow's Wind Barrage will temporarily disable but also refresh Gust
- Very short cooldown to placing Grass Traps, mainly to prevent the immediate "Too close to another trap" message
